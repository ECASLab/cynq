Detailed Description

XRTDataMover class Provides the api from which to interact with the data buffers responsable for memory operations making use of the Xilinx runtime (XRT) the types are the following: BO (Buffer object) DMA (Direct memory allocation)

Construct a new XRTDataMover object.

This constructs a data mover that uses DMA to execute the transfers between the host and the device. Moreover, it uses XRT buffer object as memory buffers.
